Teste Manual
Experimente editar a URL manualmente:
  1. Adicione ?nome=SeuNome na URL
  2. Adicione &id=999 na URL
  3. Adicione &tipo=RG na URL
  4. Pressione ENTER e veja os valores mudarem acima!
Dica: Você pode marcar esta página nos favoritos com parâmetros, e quando abrir novamente, os parâmetros estarão lá!
Bookmarkable vs Normal
Aspecto Bookmarkable Page Normal Page
URL ✅ Limpa e previsível ❌ Criptografada/aleatória
Favoritos ✅ Pode marcar ❌ Não funciona
Compartilhar ✅ Pode compartilhar ❌ Link quebra
SEO ✅ Indexável ❌ Não indexável
Histórico ✅ Funciona bem ⚠️ Pode ter problemas
Estado ❌ Stateless ✅ Stateful
