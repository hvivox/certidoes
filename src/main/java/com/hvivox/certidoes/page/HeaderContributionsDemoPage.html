Demonstração: Header Contributions

Como componentes adicionam CSS, JavaScript e outros recursos dinamicamente

Header Contributions: Permite que componentes Wicket adicionem seus próprios recursos (CSS, JavaScript, meta tags) ao <head> da página de forma automática e desacoplada.
1. CSS Inline

CssHeaderItem.forCSS(...)

CSS adicionado diretamente via renderHead()
2. JavaScript Inline

JavaScriptHeaderItem.forScript(...)

Clique no box acima!
3. CSS "Externo"

CssResourceReference(...)

Neste exemplo, simulamos CSS externo com inline
4. JavaScript "Externo"

JavaScriptResourceReference(...)

Passe o mouse sobre o box!
5. OnDomReady Script

OnDomReadyHeaderItem.forScript(...)

Executa quando o DOM está pronto
6. OnLoad Script

OnLoadHeaderItem.forScript(...)

Executa após toda a página carregar
7. Meta Tags

MetaDataHeaderItem(...)

Meta tags adicionadas:
• description
• keywords
• author
(Veja o código fonte da página)
8. Recursos de CDN

CssHeaderItem.forUrl(...)

Exemplo de Código
Como adicionar recursos no seu componente:
@Override
public void renderHead(IHeaderResponse response) {
    super.renderHead(response);
    
    // CSS Inline
    response.render(CssHeaderItem.forCSS(
        ".my-class { color: red; }", 
        "my-css-id"
    ));
    
    // JavaScript Inline
    response.render(JavaScriptHeaderItem.forScript(
        "console.log('Hello!');", 
        "my-js-id"
    ));
    
    // CSS de arquivo externo
    response.render(CssHeaderItem.forReference(
        new CssResourceReference(MyClass.class, "style.css")
    ));
    
    // JavaScript executado após DOM carregar
    response.render(OnDomReadyHeaderItem.forScript(
        "initMyComponent();"
    ));
    
    // Recurso de CDN
    response.render(CssHeaderItem.forUrl(
        "https://example.com/style.css"
    ));
    
    // Meta tag
    response.render(new MetaDataHeaderItem(
        "description", "Minha descrição"
    ));
}
